== Description ==

NetCents WooCommerce plugin provides your customers with a smooth checkout experience to pay using cryptocurrency on your WooCommerce website.


== Setup & Installation ==

Download the plugin and follow the instructions in our knowledge base to connect your website and enable crypto payments.

https://merchant-support.net-cents.com/knowledge/ecommerce-plugin/wordpress

== Screenshots ==

./assets/netcents.png

The NetCents coin is a digital currency which can be used to pay NetCents merchants.
